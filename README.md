# Charlie Chatlin

Chat application that allows users to chat with each other in a chatroom. 

## Getting Started

All scripts should be executed from the root directory.

### Prerequisites

- Node 12.x

### Installing

```
npm run i && npm run setup
```

### Validation

```
npm run validate
```

### Deployment

```
npm run start
```
or
```
npm run build
```
